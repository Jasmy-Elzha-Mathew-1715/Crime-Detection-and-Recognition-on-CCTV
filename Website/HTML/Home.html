Deep Crime Detection

A new Crime Detection and Recognition system using Computer Vision and Deep learning, to help detect criminals from the emergence of CCTV. This application is able to detect and recognise anomalies automatically.


Model

Given an input clip, we aim to classify the observed scene to different anomaly classes. Equivalently, we want our model to output the probability that an unusual event is taking place in the input video. The output of our model is the corresponding class and certain frames of the observed scene.
